# slice_* can use data masking pronouns

    Code
      lf %>% slice_max(x)
    Output
      <SQL>
      SELECT `x`, `id`
      FROM (
        SELECT `df`.*, RANK() OVER (ORDER BY `x` DESC) AS `col01`
        FROM `df`
      ) AS `q01`
      WHERE (`col01` <= 1)
    Code
      lf %>% slice_max(.data$x)
    Output
      <SQL>
      SELECT `x`, `id`
      FROM (
        SELECT `df`.*, RANK() OVER (ORDER BY `x` DESC) AS `col01`
        FROM `df`
      ) AS `q01`
      WHERE (`col01` <= 1)
    Code
      lf %>% slice_max(.data$x * .env$x)
    Output
      <SQL>

      FROM (
        SELECT `df`.*, RANK() OVER (ORDER BY `x` * -1 DESC) AS `col01`
        FROM `df`
      ) AS `q01`
      WHERE (`col01` <= 1)
