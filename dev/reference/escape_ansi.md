# Escape ANSI characters

**\[deprecated\]**

Use [`escape()`](https://dbplyr.tidyverse.org/dev/reference/escape.md)
instead.

## Usage

``` r
escape_ansi(x, parens = NA, collapse = "")
```

## Arguments

- x:

  Object to escape.

- parens, collapse:

  Controls parens and collapsing. Passed on to
  [`escape()`](https://dbplyr.tidyverse.org/dev/reference/escape.md).
