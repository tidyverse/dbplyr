# Retrieve the last SQL query generated

This is a helper function that retrieves the most recent SQL query
generated by dbplyr, which can be useful for debugging.

## Usage

``` r
last_sql()
```

## Value

A SQL string, or `NULL` if no query has been generated yet.

## Examples

``` r
library(dplyr, warn.conflicts = FALSE)

df <- lazy_frame(x = 1:3)
df |> filter(x > 1)
#> <SQL>
#> SELECT *
#> FROM "df"
#> WHERE ("x" > 1.0)

last_sql()
#> <SQL> SELECT *
#> FROM "df"
#> WHERE ("x" > 1.0)
```
